## 1. 客服岗位职责与服务规范

### 1.1 岗位职责

| 阶段 | 职责内容 |
|------|---------|
| 售前 | 主动迎宾、需求挖掘、尺码推荐、款式推荐、异议处理、促成下单 |
| 售中 | 订单确认、物流跟踪、发货通知、订单变更处理 |
| 售后 | 退换货处理、质量问题跟进、投诉处理、中差评挽回 |

### 1.2 服务规范

- **响应时效**：售前咨询 ≤ 30秒，售后问题 ≤ 15秒
- **服务态度**：热情主动，不怼客户，不推诿责任
- **回复原则**：
  - 首句必带称呼（"亲"）
  - 结尾必带感谢或祝福
  - 语气多用商量口吻，少用命令句式
  - 禁用语气词："这个不归我管"、"我不知道"、"你自己看"
- **信息记录**：每个售后工单需记录：订单号、问题描述、客户诉求、处理过程、处理结果

### 1.3 禁忌词与替代词

| 禁忌词 | 替代词 |
|--------|--------|
| 这个不归我管 | 我帮您联系相关部门处理 |
| 肯定不会坏 | 我们提供XX天质保，请放心 |
| 你自己看吧 | 我帮您详细说明一下 |
| 没办法解决 | 我帮您申请一下最优方案 |
| 退款找平台 | 我这边帮您核实退款流程 |

## 2. 售前服务流程

### 2.1 迎宾与需求挖掘

**标准开场流程：**

```
第一步：主动迎宾
  → "您好，欢迎光临XX鞋店！新款已上架，现在下单享XX优惠，很高兴为您服务~"

第二步：需求了解
  → "请问您是想选购什么类型的鞋子呢？我可以为您推荐适合的款式~"
  → "是给自己看还是帮家人朋友选购呢？"

第三步：场景深挖
  → "平时主要在什么场合穿？通勤/运动/约会/日常？"
  → "之前买鞋有没有遇到过磨脚/不透气/尺码不准的问题？"

第四步：预算了解（可选）
  → "大概什么价位比较合适？我帮您在预算内推荐性价比最高的款~"
```

### 2.2 尺码推荐

**尺码推荐原则：给建议，不给保证**

```
> "亲，尺码推荐是给您参考建议，最终由您自己决定哦~"
```

**标准尺码话术：**

| 场景 | 话术 |
|------|------|
| 第一次购买 | "平时运动鞋穿39码的话，这款建议也选39码。脚背高、脚宽的亲建议选大一码~" |
| 偏大款 | "这款偏大一码左右，建议脚瘦买小一码，脚胖或宽买正常尺码~" |
| 偏小款 | "这款偏小半码，正常脚型建议买大一码，40码以上大两码~" |
| 换码咨询 | "收到后先试穿，不合适支持7天无理由退换，记得鞋底不要弄脏哦~" |

**尺码收集标准流程：**

1. **量脚长**：建议客户赤脚站平，标记脚后跟和最长脚趾，量取厘米数
2. **确认穿着偏好**：喜欢宽松还是贴合？
3. **确认脚型**：是否偏宽、偏胖、高脚背？
4. **推荐时间**：建议下午测量，脚部会有轻微肿胀更准确

### 2.3 款式推荐

| 客户需求 | 推荐方向 |
|---------|---------|
| 日常通勤 | 乐福鞋、低跟单鞋、小白鞋（舒适为主） |
| 商务场合 | 牛津鞋、德比鞋、高跟鞋（正式感） |
| 运动健身 | 跑步鞋、训练鞋（功能专业） |
| 户外出行 | 徒步鞋、登山鞋（防水防滑） |
| 约会晚宴 | 尖头高跟鞋、切尔西靴（时尚感） |

**二选一推荐法：**
```
"您更喜欢这款黑色的商务简约风，还是棕色的复古雕花款呢？"
"我帮您对比一下这两款：A款更透气适合春夏，B款加绒保暖适合秋冬~"
```

### 2.4 价格异议处理

| 异议类型 | 话术参考 |
|---------|---------|
| 嫌贵 | "亲，买鞋主要看您喜不喜欢，如果喜欢就很有价值~我们支持7天无理由退换还送运费险，您可以买回去试试" |
| 比价 | "咱家是品牌直供/工厂直发，没有中间差价，同等品质去专柜至少贵一倍呢" |
| 等降价 | "这款现在是新品活动价，已经是全年最低了，后面只会断码不会降价哦" |
| 求优惠 | "现在下单我可以帮您申请一张XX元优惠券，再送一双品牌袜~" |

### 2.5 犹豫不决处理

```
"亲，我看您关注这款好久了，是在纠结颜色还是担心不合适呢？"

"这款目前白色黑色库存都不多了，前面刚有客户下单了白色，再不入手可能要等补货了~"

"您顾虑的是什么？是价格、款式还是担心不合适？您说出来我帮您解决~"
```

### 2.6 下单转化

```
"我帮您确认一下：黑色/39码/1双，收货地址是默认地址对吗？"
"现在下单今天16点前发出，预计后天到货~"
"买了这双跑鞋，要不要带一双运动袜？组合购买减20元，划算很多~"
```

## 3. 售中服务流程

### 3.1 订单确认

- 下单后10分钟内推送订单确认
- 高客单价订单（≥500元）建议2小时内人工确认收货地址
- 核对：商品款式、尺码、颜色、数量、收货信息

### 3.2 物流跟踪

```
发货通知："亲，您的订单已发货，快递单号：SF1234567890，预计X号到达，请注意查收~"
物流停滞："亲，看到您的包裹在XX中转站停留超过48小时，已联系快递加急处理，给您添麻烦了~"
签收提醒："亲，显示已签收，请尽快开箱检查，如有问题联系我~"
```

**物流异常处理：**

| 异常类型 | 处理时限 | 处理方案 |
|---------|---------|---------|
| 揽收超24小时 | 立即 | 联系仓库确认，告知客户原因 |
| 运输停滞超48小时 | 立即 | 联系快递催件，每2小时更新进度 |
| 丢件 | 2小时内 | 补发或全额退款+赔偿 |
| 破损 | 收到反馈即处理 | 要求拍照→补发/退款 |

### 3.3 订单变更

| 变更类型 | 未发货 | 已发货 |
|---------|-------|-------|
| 改地址 | 直接修改 | 联系快递转寄（运费自理） |
| 换款式 | 取消重拍 | 拒收后退款重拍 |
| 取消订单 | 直接取消 | 联系快递拦截（拦截成功后退款） |
| 改尺码 | 直接修改 | 拒收后重新下单 |

## 4. 售后服务流程

### 4.1 售后问题处理优先级

```
┌─────────────────────────────────────────────────┐
│                     售后咨询                      │
│                         │                        │
│          ┌──────────────┴──────────────┐          │
│          ▼                             ▼          │
│    常规问题（15分钟内）           紧急问题（5分钟内）   │
│    - 尺码不合适                        │           │
│    - 款式不满意                  - 发错货            │
│    - 物流咨询                   - 漏发/少发          │
│    - 穿着咨询                   - 严重质量问题        │
│    - 退换货流程                       │             │
│          │                             │            │
│          └──────────────┬──────────────┘           │
│                         ▼                          │
│              升级主管（30分钟未解决）                  │
│              升级经理（1小时未解决）                   │
│              升级高层（2小时未解决）                   │
└─────────────────────────────────────────────────┘
```

### 4.2 售后沟通四步法

**第一步：共情安抚**
> "亲，非常抱歉给您带来不愉快的体验，我完全理解您的心情……"

**第二步：确认问题**
> "麻烦您拍张照片发给我，我帮您核实一下具体情况~"

**第三步：提供方案**
> "针对您的情况，我这边可以为您提供以下方案：A…… B…… 您看哪个合适？"

**第四步：跟进闭环**
> "方案已为您处理完毕，预计X天内完成，有疑问随时联系我~"

### 4.3 退货退款处理流程

```
客户发起申请
     │
     ▼
核实是否符合退货条件
     │
     ├── 符合 → 同意退货 → 客户寄回 → 仓库验收 → 退款
     │
     └── 不符合 → 沟通解释 → 提供替代方案 → 协商解决
```

**退款时效：**

| 退款类型 | 处理时效 |
|---------|---------|
| 未发货退款 | 即时处理（24小时内到账） |
| 退货退款（仓库签收后） | 48小时内验货退款 |
| 质量问题退款 | 确认后24小时内处理 |
| 仅退款（部分退款） | 12小时内处理 |

## 5. 退换货标准与流程

### 5.1 退换货条件

**可退换情形：**

| 情形 | 时限 | 费用承担 |
|------|------|---------|
| 7天无理由退换 | 签收后7天内 | 非包邮：客户承担来回运费 |
| 质量问题 | 签收后15天内 | 商家承担来回运费 |
| 发错货/漏发 | 不限（需提供证据） | 商家承担 |
| 尺码不合适 | 签收后7天内 | 客户承担运费 |

**不可退换情形：**

- 鞋底有明显磨损、穿过的痕迹
- 鞋面有折痕、变形、污渍
- 吊牌已剪、包装已损坏
- 超过7天无理由期限
- 定制款/特价款（需提前告知）

### 5.2 退换货标准话术

**同意退货：**
```
"亲，麻烦您在后台申请退货（7天无理由），退款原因请选'不喜欢/不想要'，寄回时请在包裹里放一张纸条写上订单号，谢谢配合~"
```

**同意换货：**
```
"亲，已为您开通换货通道。您将商品寄回，地址：XXX，收件人：XXX，电话：XXX。寄回时麻烦在鞋盒内附上纸条：订单号+您要换的款式尺码颜色~"
```

**拒绝退换：**
```
"亲，非常抱歉，这双鞋的鞋底已有明显磨损/吊牌已剪，确实影响二次销售了，无法为您办理退换货。建议您可以转送给尺码合适的亲友，或者您看是否帮您申请一个小额的补偿优惠券？"
```

### 5.3 退换货运费规则

| 情形 | 发出运费 | 退回运费 | 说明 |
|------|---------|---------|------|
| 买家个人原因退换 | 买家承担 | 买家承担 | 包邮产品退货运费各付一半 |
| 质量问题退换 | 商家承担 | 商家承担 | 客户垫付→退款时补回 |
| 发错货/漏发 | 商家承担 | 商家承担 | 全额承担 |
| 换货二次发出 | 买家承担 | 买家承担 | 换货运费10元放鞋盒内 |

### 5.4 退款处理流程

```
申请提交
   │
   ▼
12小时内审核
   │
   ├── 审核通过
   │     │
   │     ├── 未发货 → 即时退款
   │     │
   │     └── 已发货 → 等待退货/拦截
   │           │
   │           ├── 客户寄回 → 仓库签收 → 48小时验货 → 退款
   │           │
   │           └── 快递拦截 → 确认后退款
   │
   └── 审核拒绝
         │
         └── 注明原因 → 沟通协商
```

### 5.5 补发处理

```
发错货/漏发确认
   │
   ▼
仓库核实库存
   │
   ├── 有货 → 当天补发 → 提供新单号
   │
   └── 无货 → 协商退款或换款 → 提供差价补偿
```

## 7. 常见问题处理指南

### 7.1 尺码问题

**偏大：**
```
"亲，如果只是大一点点，建议您垫半码垫或穿厚袜子，就会合脚了~
如果实在不行，不影响二次销售可以给您换小一码~"
```

**偏小：**
```
"亲，如果只是小一点点，新鞋穿穿会松一点的，皮质会慢慢变软贴合脚型~
如果确实小很多，不影响二次销售可以给您换大一码~"
```

### 7.2 磨脚问题

```
"亲，新鞋子皮质较挺，确实需要一个磨合期，一般穿3-5天后皮质软化就很舒服了~
建议您可以：
1. 先穿厚袜子在家短时间试穿，逐步适应
2. 用湿报纸塞在磨脚处撑一晚
3. 用吹风机吹软磨脚部位
如果还是不适应，支持7天无理由退换哦~"
```

### 7.3 皮质硬

```
"亲，新鞋刚开始穿会感觉皮质偏硬，穿几次后皮质会慢慢软化，越来越贴合脚型。
好的真皮鞋都是越穿越舒服的，就像养皮具一样，需要一点点耐心哦~"
```

### 7.4 掉色/褪色

```
"亲，深色皮鞋/内里首次穿着可能会有轻微掉色，这是正常现象，不是质量问题哦~
建议：
1. 第一次穿不要配白色/浅色袜子
2. 用白醋水(1:10)轻擦内里可固色
3. 穿几次后就不会掉了~"
```

### 7.5 防滑问题

```
"亲，每款鞋子的鞋底都有防滑纹路设计，在正常地面使用没有问题。
但瓷砖/大理石等光滑地面建议还是小心行走，任何鞋类在光滑湿滑地面都有打滑风险，请谅解~"
```

### 7.6 鞋底开胶

```
"亲，实在抱歉给您添麻烦了！请拍一下开胶部位的照片给我，我帮您核实处理~
方案一：寄回我们免费维修，承担来回运费
方案二：您在当地修鞋店处理，我们报销维修费（最高X元）
您看哪个方便？"
```

### 7.7 色差问题

```
"亲，非常抱歉图片和实物让您有落差感~
不同显示器确实会存在轻微色差，我们的图片都是实物拍摄，尽力还原真实颜色~
您方便拍张实物照片给我看看吗？如果是发错颜色了，我马上给您安排换货~"
```

### 7.8 透气性

```
"亲，这款鞋子采用了透气网面/真皮材质，透气性是比较好的。
但建议不要连续穿好几天，给鞋子一个干燥恢复的时间，这样可以延长使用寿命，也更卫生哦~"
```

### 7.9 异味问题

```
"亲，新鞋从密封仓库/包装中取出，多少会有些材质本身的味道，这是正常的~
建议您放在通风处晾一两天，味道就会散去~如果是刺鼻的化学味，麻烦您联系我处理~"
```

### 7.10 响声问题

```
"亲，新鞋走路有轻微响声可能是鞋底与地面摩擦的声音，穿几天鞋底磨一磨就好了~
如果持续响声，麻烦您拍个视频发我，我找仓库核实一下~"
```

## 8. 中差评处理与投诉应对

### 8.1 中差评处理流程

```
发现中差评
     │
     ▼
30分钟内响应
     │
     ├── 联系客户 → 了解不满原因
     │     │
     │     ├── 质量问题 → 道歉+补偿+方案
     │     │
     │     ├── 服务问题 → 道歉+改进承诺+优惠券
     │     │
     │     └── 预期差 → 解释+引导修改
     │
     └── 内部复盘 → 原因分析 → 改进措施
```

### 8.2 中差评回复话术

```
"亲，非常抱歉给您带来不好的购物体验。关于您反馈的[XX问题]，我们非常重视。
安排专员跟进处理，请您注意查收私信/电话。
我们会努力改进，期待能为您提供更好的服务，感谢您的反馈~"
```

### 8.3 安抚后引导修改

```
"亲，感谢您愿意给我们改进的机会！
我们已经为您申请了[XX补偿方案]。
如果您觉得我们的处理还算满意，方便的话麻烦您帮忙修改一下评价，这对我们小店真的很重要~
再次向您道歉，欢迎下次光临，一定给您更好的体验！"
```

### 8.4 投诉升级处理

|------|------|---------|

| L2 主管 | 超出权限/客户不满 | 升级处理，30分钟内响应 |
| L3 经理 | 索赔/舆情风险 | 48小时内出具方案 |
| L4 高层 | 重大客诉/法律风险 | 24小时内专项处理 |

## 9. 客服KPI考核标准

### 9.1 核心指标

| 指标 | 定义 | 目标值 |
|------|------|--------|
| 首次响应时间 | 客户首次咨询到客服回复时长 | ≤30秒 |
| 平均响应时间 | 全会话平均单次回复时长 | ≤45秒 |
| 客户满意度（CSAT） | 好评率 | ≥98% |
| 一次性解决率（FCR） | 一次咨询解决比例 | ≥85% |
| 转化率 | 咨询→下单比例 | ≥25% |
| 客单价 | 单客户平均成交金额 | 视品类定 |
| 退款率 | 退款订单/总订单 | ≤5% |
| 投诉率 | 投诉单/总订单 | ≤1% |

### 9.2 日常必做清单

**每日开店：**
- 查看前一日未回复留言
- 查看未付款订单（催付）
- 查看待发货/待处理售后

**营业中：**
- 实时回复客户咨询
- 处理售后申请（15分钟内）
- 跟进物流异常订单

**每日关店：**
- 检查当日售后处理进度
- 查看评价情况（关注差评）
- 整理当班未了事宜交接
