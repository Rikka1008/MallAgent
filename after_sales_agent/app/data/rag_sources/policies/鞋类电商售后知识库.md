# 鞋类电商售后知识库

## 一、退换货标准

### 1.1 可退换情形

| 情形 | 时限 | 运费承担 |
|------|------|---------|
| 7天无理由退换 | 签收后7天内 | 买家承担（包邮产品各付一半） |
| 质量问题 | 签收后15天内 | 商家全额承担 |
| 发错货 / 漏发 | 不限时（需证据） | 商家全额承担 |
| 尺码不合适 | 签收后7天内 | 买家承担 |

### 1.2 不可退换情形（判断标准）

以下任一情况出现，则**无法办理退换**，需向客户解释并协商替代方案：

- 鞋底有明显磨损痕迹或穿过痕迹
- 鞋面出现折痕、变形、污渍
- 吊牌已剪、包装盒已损坏
- 签收超过7天（无理由退换）或15天（质量问题）
- 商品为已告知的定制款或特价款

### 1.3 退换货操作指引

**同意退货 → 告知客户：**
> "亲，麻烦您在后台提交退货申请，退货原因请选'不喜欢/不想要'。寄回时请在包裹里放一张纸条写上您的订单号，收到后48小时内为您验货退款~"

**同意换货 → 告知客户：**
> "亲，已为您开通换货通道。请寄回至以下地址：XXX省XXX市XXX区XXX路XX号，收件人：XXX，电话：XXX。寄回时请在鞋盒内附纸条写明：订单号 + 您要换的款式/尺码/颜色~"

**拒绝退换 → 协商话术：**
> "亲，非常抱歉，这双鞋的[鞋底已有明显磨损 / 吊牌已剪]，确实影响了二次销售，无法为您办理退换。建议您考虑转送亲友，或者我帮您申请一张小额补偿优惠券，您看可以吗？"

### 1.4 运费承担规则速查

| 原因 | 发出运费 | 退回运费 |
|------|---------|---------|
| 买家个人原因 | 买家 | 买家（包邮品各半） |
| 质量问题 | 商家 | 商家（客户垫付后退款时补回） |
| 发错货/漏发 | 商家 | 商家 |
| 换货二次发出 | 买家 | 买家（换货运费10元放鞋盒内退回） |

### 1.5 退款时效

| 类型 | 时效 |
|------|------|
| 未发货退款 | 即时处理，24小时内到账 |
| 退货退款（仓库签收后） | 48小时内验货并退款 |
| 质量问题退款 | 确认后24小时内处理 |
| 仅退款（部分退款） | 12小时内处理 |

### 1.6 补发流程

1. 确认发错货 / 漏发 → 向客户索要照片证据
2. 核实仓库库存 → 有货当天补发，提供新快递单号 → 无货则协商退款或换款，提供差价补偿

## 二、物流异常处理

| 异常类型 | 处理标准 | 处理方案 |
|---------|---------|---------|
| 揽收超24小时 | 立即联系仓库确认 | 告知客户原因，承诺更新 |
| 运输停滞超48小时 | 联系快递催件 | 每2小时更新一次进度 |
| 丢件 | 2小时内给出方案 | 补发或全额退款 + 赔偿 |
| 破损 | 收到反馈即处理 | 要求拍照 → 补发或退款 |

**订单变更（已发货状态）：**

| 变更类型 | 处理方式 |
|---------|---------|
| 改地址 | 联系快递转寄（运费客户自理） |
| 换款式/尺码 | 告知客户拒收后重新下单 |
| 取消订单 | 联系快递拦截，拦截成功后安排退款 |

## 四、常见售后问题应对方案

### 4.1 尺码偏大

> "亲，如果只是大一点点，建议垫半码垫或穿厚袜子就会合脚了。如果确实大很多，不影响二次销售的情况下可以给您换小一码哦~"

### 4.2 尺码偏小

> "亲，新鞋穿穿会松一点的，皮质会慢慢变软贴合脚型。如果只是小一点点，建议再试穿适应一下。如果确实小很多，不影响二次销售的情况下可以给您换大一码~"

### 4.3 磨脚

> "亲，新鞋皮质较挺，确实需要一个磨合期，一般穿3-5天后皮质软化就很舒服了。建议您：
> 1. 先穿厚袜子在家短时间试穿，逐步适应
> 2. 用湿报纸塞在磨脚处撑一晚
> 3. 用吹风机吹软磨脚部位
> 如果试过还是不适应，支持7天无理由退换哦~"

### 4.4 皮质偏硬

> "亲，新鞋刚开始穿会感觉皮质偏硬，穿几次后就会慢慢软化，越来越贴合脚型。好的真皮鞋都是越穿越舒服的，就像养皮具一样需要一点点耐心哦~"

### 4.5 掉色 / 褪色

> "亲，深色皮鞋内里首次穿着可能会有轻微掉色，这是正常现象，不属于质量问题哦。建议前几次不要配浅色袜子，用白醋水(1:10)轻擦内里可以固色，穿几次后就不会掉了~"

**判断标准：** 轻微浮色属正常；大面积脱色或染色到脚上 → 按质量问题处理。

### 4.6 防滑质疑

> "亲，每款鞋底都有防滑纹路设计，正常地面使用没有问题。但在瓷砖、大理石等光滑地面上建议小心行走，任何鞋类在湿滑地面都有打滑风险，这点还请您理解~"

### 4.7 鞋底开胶（质量问题）

> "亲，实在抱歉给您添麻烦了！麻烦拍一下开胶部位的照片发给我，我帮您核实处理~
> 方案一：寄回我们免费维修，来回运费我们承担
> 方案二：您在当地修鞋店处理，我们报销维修费（最高X元）
> 您看哪个方便~"

**定责标准：** 开胶长度 ≤ 2cm 且为新鞋 → 按质量问题处理；开胶面积大或穿着超过3个月 → 补偿方案协商。

### 4.8 色差

> "亲，非常抱歉图片和实物让您有落差感。不同显示器确实会存在轻微色差，我们的图片都是实物拍摄，尽力还原真实颜色的。您方便拍张实物照片给我看看吗？如果是发错颜色了，我马上给您安排换货~"

**定责标准：** 轻微色差属正常范围；颜色完全不对 → 核实是否发错货，是则商家承担退换。

### 4.9 透气性

> "亲，这款鞋子采用了[XX材质]，透气性是比较好的。不过建议不要连续穿好几天，给鞋子干燥恢复的时间，这样可以延长使用寿命也更卫生哦~"

### 4.10 异味

> "亲，新鞋从密封包装中取出多少会有些材质本身的味道，这是正常的。建议放在通风处晾一两天味道就会散去。如果是刺鼻的化学味，麻烦您联系我，我帮您特殊处理~"

### 4.11 走路响声

> "亲，新鞋走路有轻微响声可能是鞋底与地面摩擦的声音，穿几天鞋底磨一磨就好了。如果持续响声较大，麻烦拍个视频发我，我找仓库核实一下~"

## 五、投诉与差评处理

### 5.1 处理原则

- 30分钟内首次响应
- 先了解原因，再提供方案，最后引导修改评价
- 不推卸责任，即使是客户的误解也要委婉解释
- 所有差评处理必须记录工单并内部复盘

### 5.2 差评回复话术

> "亲，非常抱歉给您带来不好的购物体验。关于您反馈的问题我们非常重视，已安排专员跟进处理。我们会努力改进，期待为您提供更好的服务，感谢您的反馈~"

### 5.3 安抚后引导修改评价

> "亲，感谢您愿意给我们改进的机会！我们已经为您申请了[XX补偿方案]。如果您觉得处理还算满意，方便的话麻烦修改一下评价，这对我们小店真的很重要~ 再次向您道歉，欢迎下次光临！"

### 5.4 升级机制

| 级别 | 触发条件 | 处理方式 |
|------|---------|---------|
| L1 - AI客服 | 标准售后问题 | 按知识库直接处理并记录 |
| L2 - 人工主管 | 超出权限 / 客户不满 / 威胁差评 | 升级工单，30分钟内人工响应 |
| L3 - 经理 | 索赔需求 / 舆情风险 | 48小时内出具方案 |
| L4 - 高层 | 重大客诉 / 法律风险 | 24小时内专项处理 |

**升级话术：**
> "亲，您的问题我已经详细记录了，为了让您得到更专业的处理，我帮您转接给售后主管，稍后会有专人联系您，请保持电话畅通~"
